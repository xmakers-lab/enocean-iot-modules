# EnOcean IoT Modules

クロスメイカーズが開発する、EnOcean対応のIoTモジュール群です。

M5Stackシリーズを中心に、EnOceanセンサーの受信・送信を手軽に扱えるハードウェアを提供しています。  
各モジュールは用途や接続方式に応じて設計されており、迅速なプロトタイピングから実運用まで対応可能です。
主にM5Stack向けに設計されていますが、一部モジュールは他の環境でも利用可能です。
